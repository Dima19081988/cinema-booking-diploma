# Дипломный проект по профессии «Веб-разработчик: Backend-разработка на Python»
# Тема: **Cайт-агрегатор просмотра и бронирования билетов**

Дипломный проект представляет собой создание сайта для бронирования онлайн билетов в кинотеатр и разработка информационной системы для администрирования залов, сеансов и предварительного бронирования билетов.

## Задачи

* Разработать сайт бронирования билетов онлайн.
* Разработать административную часть сайта.
* В результате должно получиться комплексное веб-приложение, включающее:
    - бэкенд на языке **Python** с использованием фреймворка **Django** или **FastAPI** и СУБД **PostgreSQL**;
    - фронтенд на языках **JavaScript**, **HTML**, **CSS** с использованием библиотек **React**, **React Router**.
* В работе над интерфейсом используйте разработанные **UI**-компоненты системы ([вёрстка](./sources/layouts.zip)).

## Функциональные возможности

### Возможности гостя
- Просмотр списка фильмов
- Просмотр расписания сеансов
- Выбор мест в кинозале
- Бронирование билета на конкретную дату и сеанс

### Возможности администратора
- Создание и редактирование кинозалов
- Создание и редактирование фильмов
- Настройка цен
- Создание и редактирование расписания сеансов
- Просмотр броней

## Технологический стек

### Backend
- Python
- Django
- Django REST Framework
- PostgreSQL

### Frontend
- React
- React Router
- JavaScript
- HTML
- CSS

## Архитектурная идея
БД: PostgreSQL
Backend-часть: Django + Django REST Framework.
Frontend-часть: React + React Router.
Генерация билетов с QR-кодом: Python-библиотека `qrcode`.
